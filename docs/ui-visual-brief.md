# UI Visual Brief — sound escape

## Visual Concept（一文）

**「停電した深夜のホームセンターへ投入した回収ロボットを、暗闇の中で音波ソナーだけを頼りに遠隔操作する“夜間遺失物回収局”の管制ターミナル」**

潜水艦ソナー卓 × 産業用ロボット遠隔操作端末 × 古いCRTレーダーの質感を混ぜる。プレイヤーは「画面の中の世界」ではなく「現場へ送り込んだ機械の管制官」として座っている感覚を持つ。

## 参照する形・材質・時代

- 形: 円形スコープ、同心円スケール、方位目盛り、スイープ光、矩形のメタルパネル、ビス留めのベゼル、刻印ラベル、ロッカースイッチ。
- 材質: 黒く塗装された金属、すり減った塗膜、薄い埃、CRTの走査線とビネット、ガラス面の反射。
- 時代: 80〜90年代の業務用機器を、現代のミニマルなUIで再構成した「ネオ・レトロ計器」。

## 配色（役割）

| 役割 | 色 | 用途 |
|---|---|---|
| base | `#04060A`（ほぼ黒） / `#0A1410`（深緑がかった黒） | 背景・パネル地 |
| accent（音波） | `#39FF9E`〜`#16C76A`（ネオングリーン） | 音波・地形輪郭・主操作 |
| item（忘れ物） | `#FFD34E`（琥珀） / `#EAFFB0`（白緑） | 忘れ物の二重反響 |
| danger（幽霊/危険） | `#FF3D6E`〜`#B026FF`（赤紫） | 幽霊シルエット・危険床・被弾 |
| text | `#CFF7E0`（淡い蛍光緑） / `#7Fae98`（減光） | ラベル・本文 |

色だけに意味を持たせない：忘れ物=点滅+二重円、危険=脈動+形状、幽霊=ノイズ形状で区別する。

## background / control-skin / motif の用途

- **background**: タイトルとプレイ画面のスコープ背景。中央はUIオーバーレイ用に余白を確保し、四隅にビネット。文字は焼き込まない。
- **control-skin**: 主操作ボタン（PING / 回収開始）の文字なし金属フレーム。中央は伸ばしても破綻しないフラットなサーフェス、明確なエッジと押し込み表現。ラベルはHTML側。
- **motif**: 局のエンブレム（ソナー波＋忘れ物アイコンを内包した円形バッジ）、コーナー装飾、ディバイダ。

## overlay / contrast / crop / mobile

- 背景中央は計器を重ねるため低コントラスト・暗めに保つ。
- 横長背景は16:9想定。mobile（390px）では縦に計器を積むため、背景は object-cover で中央クロップに耐える構図。
- 主要テキストは蛍光緑で AA コントラスト確保。

## 禁止するAIテンプレ表現

- 紫青グラデの主役化 / 意味のないガラスカード / ランダム発光blob / 汎用AIアシスタント感 / SaaSダッシュボード / 過剰blur。
- 「録音中」と誤解させる赤丸表示（要件で禁止）。
- 画像への文字焼き込み。
